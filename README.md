# MP Preview

一键将 Markdown 文档转换为微信公众号格式插件。

## 功能特点

- 一键预览 Markdown 文件的公众号排版效果
- 实时预览更新
- 自定义模板和字体
- 便捷的复制功能

## 安装方法

1. 打开设置
2. 转到第三方插件设置，关闭安全模式
3. 点击浏览并搜索 "MP Preview"
4. 安装并启用插件

## 使用方法

1. 打开任意 Markdown 文件
2. 点击侧边栏的预览图标
3. 自定义预览设置
4. 复制格式化后的内容
